# Airport Activity Explorer 

A simple web app that shows what’s happening at different airports using live aviation data. Instead of digging through raw data, you can just search an airport and quickly see arrivals, departures, and general activity.

---

## What this does

You search for an airport, and the app pulls data from an aviation API and shows it in a clean way.

You can:

* See incoming and outgoing flights
* Get a feel for how busy an airport is
* Notice patterns like peak hours or frequent airlines

---

## Features

* Search airports by name or code
* View arrivals and departures
* Basic charts for busy times (optional)
* Simple, easy-to-read layout

---

## How it works

1. Enter an airport
2. The app fetches data from the API
3. It cleans and organizes the data
4. You see the results in a readable format

---

## Tech used

* HTML, CSS, JavaScript
* Aviation API (like OpenSky or Aviationstack)
* (Optional) Chart library for graphs
